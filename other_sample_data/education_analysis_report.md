# 教育データ総合解析レポート

## 概要

5つの教育関連オープンデータセットをダウンロード・解析し、探索的データ分析（EDA）と機械学習による予測モデルの構築を行いました。

---

## Dataset 1: Kaggle Students Performance Dataset

**データソース**: https://www.kaggle.com/datasets/rabieelkharoua/students-performance-dataset

| 項目 | 値 |
|------|-----|
| レコード数 | 2,392人 |
| 変数数 | 15 |
| 対象 | 高校生 |
| 欠損値 | なし |

### 主要な変数
- `Age`: 年齢（15〜18歳）
- `Gender`: 性別
- `StudyTimeWeekly`: 週間勉強時間
- `Absences`: 欠席回数
- `ParentalSupport`: 親のサポートレベル（0〜4）
- `GPA`: 成績（0〜4.0）
- `GradeClass`: 成績クラス（0〜4）

### 主要な知見
- **平均GPA**: 1.91（標準偏差 0.92）
- **勉強時間とGPA**: 正の相関が確認された
- **親のサポート**: サポートレベルが高いほどGPA向上の傾向
- **欠席と成績**: 欠席回数が増えるとGPA低下

### 機械学習モデル（Random Forest回帰）
- **R² = 0.927**: 非常に高い予測精度
- **RMSE = 0.246**
- **最も重要な特徴量**: 欠席回数、勉強時間、親のサポート

## Dataset 2: UCI Student Performance

**データソース**: https://archive.ics.uci.edu/dataset/320/student+performance

| 項目 | 値 |
|------|-----|
| レコード数 | 649人 |
| 変数数 | 33 |
| 対象 | ポルトガルの中等教育 |

### 主要な変数
- `studytime`: 勉強時間（1=<2h, 2=2-5h, 3=5-10h, 4=>10h）
- `Medu`/`Fedu`: 母/父の教育レベル（0〜4）
- `Dalc`/`Walc`: 平日/週末のアルコール消費量
- `failures`: 過去の不合格回数
- `G1`, `G2`, `G3`: 第1期・第2期・最終成績（0〜20点）

### 主要な知見
- **最終成績(G3)平均**: 11.9点（20点満点）
- **中間成績(G1,G2)**: 最終成績の最強予測因子（当然ながら）
- **親の教育レベル**: 高いほど成績良好（特に母親の影響が大きい）
- **アルコール消費**: 消費量増加で成績低下の明確な傾向
- **過去の不合格**: 不合格経験が多いほど最終成績が大幅に低下

### 機械学習モデル（Random Forest回帰）
- **R² = 0.841**: 高い予測精度
- **RMSE = 1.244**
- **最重要特徴量**: G2（第2期成績）> G1 > failures > absences

## Dataset 3: xAPI-Edu-Data（学習行動データ）

**データソース**: https://www.kaggle.com/datasets/aljarah/xAPI-Edu-Data

| 項目 | 値 |
|------|-----|
| レコード数 | 480人 |
| 変数数 | 17 |
| 対象 | LMS利用学生 |

### クラス分布
- High(H): 142人（29.6%）
- Middle(M): 211人（44.0%）
- Low(L): 127人（26.5%）

### 主要な変数
- `raisedhands`: 挙手回数
- `VisITedResources`: 学習リソース閲覧数
- `AnnouncementsView`: お知らせ閲覧数
- `Discussion`: ディスカッション参加数
- `StudentAbsenceDays`: 欠席日数カテゴリ

### 主要な知見
- **挙手回数**: 高成績クラスほど挙手回数が多い（明確な差）
- **リソース閲覧**: 高成績者は平均60回以上、低成績者は30回以下
- **欠席日数**: 7日以上欠席の学生は低成績クラスに集中
- **親のアンケート回答**: 回答する親の子供は成績が良い傾向

### 機械学習モデル（Random Forest分類）
- **Accuracy = 0.854**: 良好な分類精度
- **Low(L)の再現率**: 0.96（最も高い）
- **最重要特徴量**: VisITedResources > raisedhands > Discussion > AnnouncementsView

## Dataset 4: 文部科学省パブリックユースデータ（令和6年度）

**データソース**: https://www.mext.go.jp/a_menu/shotou/gakuryoku-chousa/sonota/1404609.htm

| 項目 | 児童（小学校） | 生徒（中学校） |
|------|---------------|---------------|
| レコード数 | 10,000人 | 10,000人 |
| 変数数 | 56 | 57 |
| 教科 | 国語・算数 | 国語・数学 |

※擬似データ（統計的性質を保存した匿名データ）

### 主要な知見
- **小学校**: 国語と算数の正答数に相関がある
- **中学校**: 国語と数学の正答数にも相関あり
- **性別差**: 性別による平均正答数の差が観察された
- **地域規模**: 地域規模による成績差がデータに現れている
- **問題別分析**: 問題ごとの正答率にばらつきがあり、難問と易問が明確

### データの特徴
- 個別の問題レベルの正誤（正誤_国_01〜14、正誤_算_01〜16等）
- 児童・生徒質問回答データ（学習習慣等に関するアンケート）
- 地域規模・学校番号による集計が可能

## Dataset 5: SSDSE（教育用標準データセット）

**データソース**: https://www.nstac.go.jp/SSDSE/index.html

### SSDSE-A（市区町村別）
| 項目 | 値 |
|------|-----|
| レコード数 | 1,741市区町村 |
| 変数数 | 128 |
| 分野 | 人口・経済・教育・医療等 |

### SSDSE-B（県別推移）
| 項目 | 値 |
|------|-----|
| レコード数 | 565（47都道府県×12年程度） |
| 変数数 | 112 |

### SSDSE-E（基本素材）
| 項目 | 値 |
|------|-----|
| レコード数 | 48（全国＋47都道府県） |
| 変数数 | 95 |
| 教育関連変数 | 幼稚園数/在園者数、小学校数/児童数、中学校数/生徒数、高等学校数/生徒数、大学数/学生数等 |

### 主要な知見
- **人口の偏り**: 東京都が突出（約1,240万人）、市区町村レベルでは大きな偏り
- **少子高齢化**: 年少率（15歳未満）と高齢化率（65歳以上）に強い負の相関
- **合計特殊出生率**: 都道府県間で1.12〜1.70程度の幅
- **時系列トレンド**: 全国で少子化・高齢化が一貫して進行
- **教育インフラ**: 都道府県ごとの学校数・児童生徒数のデータで地域間格差を分析可能

## 全体のまとめ・クロスデータセットの知見

### 共通して見られたパターン

1. **出席・参加の重要性**: すべてのデータセットで、出席率・参加度と成績に正の関係
2. **家庭環境の影響**: 親の教育レベル・サポートが成績に影響（Dataset 1, 2, 3）
3. **学習行動が鍵**: 勉強時間・リソース閲覧・挙手回数等のアクティブな学習行動が成績を左右
4. **地域格差**: 日本のデータ（Dataset 4, 5）では地域規模・都道府県による差が存在
5. **予測可能性**: Random Forestモデルで高い予測精度を達成（R²=0.84〜0.93）

### 今後の発展可能性

- より高度な機械学習モデル（XGBoost, Deep Learning等）の適用
- データセット間のメタ分析
- 日本のSSDSEデータと文科省データの結合分析
- 時系列分析による教育トレンドの予測
- 因果推論手法の適用（傾向スコア、操作変数法等）
