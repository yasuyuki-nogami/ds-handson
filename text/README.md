# データサイエンス・ハンズオン テキスト（全4章）

岡山県 高校教員向け・3時間ワークショップの**読み物版（テキスト）**です。
ノートブック（`notebooks/`）の内容を、上から読める4章構成の Markdown にまとめています。
Chromebook + Google Colab だけで動きます（インストール不要・無料）。

## 目次

| 章 | 内容 | 位置づけ |
|---|---|---|
| [第1章 基礎編](01_基礎編.md) | Python超入門／pandasで公開データを読む／集計・並べ替え／matplotlibで可視化 | 1時間目（座学＋基礎）|
| [第2章 応用ハンズオン](02_応用ハンズオン.md) | 群間比較・相関・相関行列・単回帰／相関≠因果 | 2時間目（自分で手を動かす）|
| [第3章 グループワーク](03_グループワーク.md) | 「問い→データ→分析→わかったこと→注意」で自由に分析（完成例つき）| 3時間目（グループ自由演習）|
| [第4章 発展編](04_発展編.md) | クラスタリング(K-means)／主成分分析(PCA)／回帰予測／因果推論 | 発展・デモ（実行するだけ）|
