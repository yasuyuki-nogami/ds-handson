# データ説明書（データパック）

演習で使う4つのCSVの説明です。すべて `data/` フォルダに入っています。
ノートブックでは `load("ファイル名.csv")` で読み込めます。

---

## 1. prefecture.csv（実データ）
47都道府県の基本統計。

| 列名 | 意味 |
|---|---|
| code | 都道府県コード（1〜47, JIS順）|
| prefecture | 都道府県名 |
| region | 地方区分（北海道/東北/関東/中部/近畿/中国/四国/九州・沖縄）|
| population_2020 | 人口（2020年 国勢調査, 人）|
| area_km2 | 総面積（km²）|
| density_per_km2 | 人口密度（人/km²、population_2020 ÷ area_km2 で算出）|

- 出典：総務省統計局「国勢調査（2020年）」、面積は国土地理院/総務省資料。
- 用途：ランキング、相関（面積×人口）、地方別集計、クラスタリング。

## 2. school_health.csv（実データを整理）
年齢別・男女別の平均身長・体重。

| 列名 | 意味 |
|---|---|
| age | 年齢（5〜17歳）|
| sex | male / female |
| height_cm | 平均身長（cm）|
| weight_kg | 平均体重（kg）|

- 出典：文部科学省「学校保健統計調査（令和5年度）」年齢別 平均身長・平均体重をもとに整理。
- 注意：概数として整理しています。授業で正確な最新値を使う場合は e-Stat で確認してください。
- 用途：折れ線（成長曲線）、散布図・相関（身長×体重）、BMI計算、単回帰。

## 3. gakuryoku_lifestyle.csv（実データを整理）
生活習慣の「群」ごとの平均正答率。

| 列名 | 意味 |
|---|---|
| category | 生活習慣の種類（朝食/睡眠/スマホ・ゲーム/読書）|
| group | 群（例：毎日食べる 等）|
| japanese_score | 中3 国語の平均正答率(%)（イメージ値）|
| math_score | 中3 数学の平均正答率(%)（イメージ値）|

- 出典：国立教育政策研究所「全国学力・学習状況調査」の質問紙×正答率クロス集計の傾向をもとに整理。
- 注意：年度により実数値は変わります。傾向（良い生活習慣ほど正答率が高い）を体験する教材用です。
- 用途：群間比較（棒グラフ）、「相関≠因果」の議論。

## 4. class_sample.csv（架空のサンプル・個人情報ではありません）
ある高校の生徒360人ぶんの練習用データ。**プログラムで生成した架空データ**です。

| 列名 | 意味 |
|---|---|
| student_id | 生徒番号 |
| grade | 学年（1〜3）|
| sex | male / female |
| height_cm, weight_kg | 身長・体重 |
| study_min | 1日の勉強時間（分）|
| sleep_hours | 睡眠時間（時間）|
| breakfast | 朝食（1=食べる, 0=食べない）|
| smartphone_hours | スマホ利用時間（時間）|
| test_score | テストの点数（0〜100）|

- 用途：相関・回帰・群間比較・相関行列を「個票（1人ずつの行）」で安全に練習。
  実在データではないのでプライバシーの心配なく自由に使えます。

## 【公式実データ】e-Stat API から実際にダウンロードしたファイル（`data_official/`）

`data/` の教材用CSVとは別に、**政府統計の総合窓口 e-Stat の公式API から実際に取得した実データ**を
`data_official/` に別ファイル名で保存しています（既存の教材ファイルは変更していません）。
再取得は `scripts/fetch_estat.py` を実行すれば誰でも再現できます（要 e-Stat の無料アプリケーションID）。

```bash
export ESTAT_APP_ID=あなたのアプリケーションID   # https://www.e-stat.go.jp/api/ で無料発行
python3 scripts/fetch_estat.py
```

| ファイル | 内容 | 出典（statsDataId）| 取得日 |
|---|---|---|---|
| `data_official/prefecture_estat2020.csv` | 47都道府県 人口・面積・人口密度 | 総務省統計局「令和2年国勢調査 人口等基本集計」(0003445099) | 2026-07-18 |
| `data_official/school_health_estat2019.csv` | 年齢別・男女別 平均身長・平均体重（2019年度）| 文部科学省「学校保健統計調査 平成27年度以降 全国表」(0003146500) | 2026-07-18 |
| `data_official/raw/*.json` | 上記の**APIの生レスポンス**（無加工の記録用）| e-Stat API v3.0 | 2026-07-18 |

- **prefecture_estat2020.csv**：人口=「2015年人口＋5年間の人口増減数」で復元（＝2020年総人口）。全国計は 126,146,099 人で公式値と一致。
  `density_per_km2` は e-Stat の**公表値**（例：北海道は北方領土を除く面積で算出されるため、人口÷面積とは一致しません）。
  ※既存の `data/prefecture.csv` は「人口÷面積」で密度を算出しており、人口の値自体は両者とも同一です。
- **school_health_estat2019.csv**：設置者=計、都市階級=計（全国）、平均値。API上の最新年度が2019年度のため2019年度値です。
- **出典表記の例**：「出典：政府統計の総合窓口(e-Stat)、総務省統計局『令和2年国勢調査』／文部科学省『学校保健統計調査』を加工して作成」

> 全国学力・学習状況調査（`gakuryoku_lifestyle.csv` の元ネタ）は **e-Stat API では配信されていない**ため、
> API での自動取得はできませんでした。実数値が必要な場合は国立教育政策研究所の公表資料（Excel）から手作業での取り込みが必要です。

## データの配り方（Chromebook / Colab 向け）
おすすめは次のどちらか。

**方法A：GitHub 公開（最もスムーズ）**
1. このフォルダごと GitHub の公開リポジトリに置く。
2. 各ノートの `BASE_URL` を `https://raw.githubusercontent.com/ユーザー名/リポジトリ名/main/data/` に書き換える。
3. 受講者は Colab で「GitHubから開く」→ ノートを実行するだけ。アップロード不要。

**方法B：手動アップロード（準備不要）**
- `BASE_URL` はそのままでOK。`load()` が自動で「アップロードしてください」と促します。
- 受講者は配布された CSV を選んでアップロードすれば読み込めます。
